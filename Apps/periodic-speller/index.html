How it works

Each letter or pair of letters is matched against the 118 element symbols from the periodic table. The algorithm tries two-letter symbols first (like He , Li , Ac ) then falls back to single-letter ones ( B , C , N , etc.). If a full decomposition can't be found, unmatched letters appear as clickable gray tiles — click one to create a custom fake element for that symbol.
