executable folder contains all necessary files to run program
source file contains all necessary files to compile program (aka proof that we wrote it ;)

run with Pacman.exe (only works in Windows....)

***********************

Controls for game mode:

Page Up & Page Down: zoom in / out on pacman
~ : 	switch to debug mode

Controls change depending on camera mode:

Birds Eye View:
up, down, left, right arrows : pacman movement

Birds Eye Follow:
use mouse to direct pacman
use arrow keys to move toward facing direction

Third Person Point of View:
use left and right keys to rotate pacman
use up and down keys to move pacman

First Person Point of View:
use mouse to rotate pacman
use up and down keys to move pacman

***********************

Controls for debug mode:

~ : 	switch to game mode
F1 : 	change camera mode 
(Birds Eye, Birds Eye Follow, First Person Point of View, Third Person Point of View)
1 & 3 : change z coordinates
Page Up & Page Down : change y coordinates
Home & End : change x coordinates
4 & 6 : rotate around z axis
7 & 9 : rotate around y axis
8 & 5 : rotate around x axis
x : turn on and off map rotation
f : full screen
z : reset changes to default
Esc : quit
